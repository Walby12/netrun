# netrun

**netrun** miny cyberpunk enviroment built from scratch

> [!WARNING]  
> **Status: Early Development.** netrun is currently an experimental project. The internal architecture and API are subject to frequent, breaking changes.

## Prerequisites

To build and run **netrun**, you need:

1.  **C Compiler:** `gcc` or `clang` (C99/C11 support required).
2.  **GPU Hardware:** Graphics card.
3.  **Internet Access:** (Planned for future releases).
